# 30-day plan — money first

| Day | Action | Done |
|-----|--------|------|
| 1 | Replace placeholders in `marketing/landing/index.html`, deploy to Vercel (free) | ☐ |
| 1 | Buy domain (~$12): afterline.ai or getafterline.com | ☐ |
| 2 | Set up Calendly: "15-min Revenue Leak Audit" | ☐ |
| 2 | Build list: 100 local HVAC/plumbing companies in your area (Google Maps) | ☐ |
| 3–5 | Send 25 cold emails/day using `marketing/outreach/cold-email.md` | ☐ |

## Daily non-negotiables (30 min minimum)

1. **Outbound** — never skip
2. **One delivery touch** per active client
3. **One improvement** to templates or landing page
